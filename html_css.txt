HTML 
ประกอบด้วย 3 ส่วนหลักๆ
1. <html></html> คลุมทุกส่วนเพื่อกำหนดว่าเป็น html
2. <head></head> แสดงผลบน Tab bar
3. <body></body> ส่วนเนื้อหาของเว็บไซด์

<meta charset="UTF-8"> : คือคำสั่งที่ใส่ตรงส่วน head เพื่อให้หน้าเว็บเข้าใจอักษรภาษา
<meta name="viewport" content="width=device-width, initial-scale=1.0"> : คือคำสั่งที่ใส่ตรงส่วน head เพื่อให้แสดงผลกับ smartphone ได้
<title></title> : กำหนดชื่อบน head 
<h1 style="color:red;font-size: 20px;">Title</h1> : h1 คือหัวเรื่องจะมีขนาดใหญ่ h2,h3,... จะเล็กลงตามลำดับ (*สามารถกำหนด style css บน tag ได้)
<p></p> คือ text ทั่วไป
<img src="แปะเป็น link หรือชื่อไฟล์ .jpg ก็ได้" style="width: 10%";> : กำหนดขนาดของรูปเพิ่มเติม
<a href="link ที่จะให้เชื่อมต่อหรือเชื่อมต่อ webpage html อีกไฟล์">TEST</a> : กำหนดข้อความ "TEST" ถ้าคลิ๊กจะเป็น link
<div></div> : เป็นการสร้างกรอบสามารถใส่ style เพิ่มเพื่อให้เห็นกรอบ
<input></input> : สร้างกล่องกรอกข้อความ
<button></button> : สร้างปุ่มกด
<header></header> : เพื่อเป็นกรอบในการสร้าง Menu ด้านบนเว็บ (คือ Div)
<footer></footer> : เพื่อเป็นกรอบในการสร้าง Menu ด้านล่างเว็บ (คือ Div)
//ตาราง table
<table>
<thead> ส่วนหัวตาราง
<tbody> ส่วนใส่ข้อมูลตาราง
<tfoot> ส่วนล่างตาราง
<tr> ใช้เก็บ th (หัวตารางเพราะมีความหนา) (ใส่ใน thead)
<tr> ใช้เก็บ td (ช่องตารางใช้เก็บข้อความธรรมดา) (ใส่ใน tbody)
การขยายช่องตาราง (merge) : (*ต้องลบช่องตารางที่ไม่ใช้ก่อน)
- <th colspan="2"> (เท่ากับกำหนดว่าจะกินพื้นที่แนวคอลัมน์ 2ช่อง)
- <th rowspqn="2"> (เท่ากับกำหนดว่าจะกินพื้นที่แนวแถว 2ช่อง) (*โดยมันจะผลักช่องถัดไปออกให้เอง)
- <th colspan="2" rowspqn="2"> (เท่ากับกำหนดว่าจะกินพื้นที่แนวคอลัมน์และแนวแถวรวม 4ช่อง)

*ใส่ style เพิ่มใน tag บางครั้งโค้ดจะยาวเกินควรสร้าง css มาช่วย

CSS
<link rel="stylesheet" href="ชื่อไฟล์ css"> : เป็นการ import ให้ไฟล์ css เข้ามาในไฟล์ html (ใส่บน head ของ html)
div {background-color: aqua; padding(พื้นที่ภายในกรอบ): 20px;} : จะเป็นกำหนดสีหรือฟ้อนต่างๆลงใน tag div บนโค้ด html (แต่จะหมายถึง div ทุกตัวในไฟล์ html)
class="ตั้งชื่อ" ใน tag บน html เช่น <div class="box"> เพื่อนำมาใช้ภายใน css ตัวอย่าง
    .box{background-color: blue;font-size:10px;}
.button .message(คือการกำหนดให้ทั้ง2class เปลี่ยนตามที่กำหนด) {
    color: white; *สีของข้อความ
    background-color: black; *สีพื้นหลัง
    padding: 10px; *พื้นที่ภายในกรอบ
    text-align: center; *ตำแหน่งตัวหนังสือ
    margin-bottom: 10px; *margin = ระยะขอบ
    display: block; *ปรับการแสดงผลจะทำให้ tag จองเต็มบรรทัด , inline-box แสดงผลไม่เต็มบรรทัด
    width: 30%; *ความกว้างของปุ่ม
    text-decoration: none; *ลบขีดเส้นใต้ของปุ่มที่เชื่อมลิ้ง
    border: 4px solid(ประเภทของเส้น) rgb(60, 59, 59); *เส้นขอบมักจะกำหนดขนาดและสีไปพร้อมๆกัน
    border-color: red; *สีของเส้นขอบ
    box-shadow: 4px 4px (2ตัวแรกคือตำแหน่งของเงา) 8px 4px (ขนาดของเงา) rgb(126, 214, 126); *เงาของกรอบ
}
